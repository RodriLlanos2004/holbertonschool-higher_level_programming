# SQL - More queries

Este proyecto dentro de `holbertonschool-higher_level_programming` se enfoca en el manejo avanzado de bases de datos relacionales con MySQL, incluyendo permisos de usuario, llaves foráneas, JOINs y subconsultas.

## Tareas y Scripts

* **0-privileges.sql**: Lista todos los privilegios de los usuarios `user_0d_1` y `user_0d_2`.
* **1-create_user.sql**: Crea el usuario `user_0d_1` con todos los privilegios.
* **2-create_read_user.sql**: Crea una base de datos y un usuario con permisos de solo lectura.
* **3-force_name.sql**: Crea una tabla donde el nombre no puede ser nulo.
* **4-never_empty.sql**: Crea una tabla con un valor por defecto para el ID.
* **5-unique_id.sql**: Crea una tabla donde el ID debe ser único.
* **6-states.sql**: Crea la base de datos `hbtn_0d_usa` y la tabla `states`.
* **7-cities.sql**: Crea la tabla `cities` con una llave foránea que referencia a `states`.
* **8-cities_of_california_subquery.sql**: Lista ciudades usando subconsultas.
* **9-cities_by_state_join.sql**: Lista ciudades y estados usando un JOIN.
* **10-genre_id_by_show.sql**: Lista shows con su respectivo ID de género.
* **11-genre_id_all_shows.sql**: Lista todos los shows, mostrando el género si existe.
* **12-no_genre.sql**: Lista los shows que no tienen género asignado.
* **13-count_shows_by_genre.sql**: Cuenta cuántos shows hay por cada género.
* **14-my_genres.sql**: Lista los géneros del show 'Dexter'.
* **15-comedy_only.sql**: Lista todos los shows de comedia.
* **16-shows_by_genre.sql**: Lista todos los shows y sus géneros asociados.
