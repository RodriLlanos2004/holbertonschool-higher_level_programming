# SQL Introduction

Este proyecto marca el inicio en el mundo de las bases de datos relacionales utilizando MySQL. Aquí se gestionan bases de datos, tablas y consultas de manipulación de datos (DML).

## Lista de Tareas (Archivos SQL)

* **0-list_databases.sql**: Lista todas las bases de datos del servidor.
* **1-create_database_if_missing.sql**: Crea la base de datos si no existe.
* **2-remove_database.sql**: Borra la base de datos especificada.
* **3-list_tables.sql**: Lista todas las tablas de la base de datos actual.
* **4-first_table.sql**: Crea la primera tabla con campos ID y Nombre.
* **5-full_table.sql**: Muestra el esquema completo de la tabla.
* **6-list_values.sql**: Selecciona y muestra todos los registros de la tabla.
* **7-insert_value.sql**: Inserta un nuevo registro en la tabla.
* **8-count_89.sql**: Cuenta registros con un ID de valor 89.
* **9-full_creation.sql**: Crea una tabla e inserta múltiples valores iniciales.
* **10-top_score.sql**: Ordena registros por puntaje descendente.
* **11-best_score.sql**: Filtra registros con puntaje mayor o igual a 10.
* **12-no_cheating.sql**: Corrige/actualiza el puntaje de un usuario.
* **13-change_class.sql**: Borra registros que no cumplen con el puntaje mínimo.
* **14-average.sql**: Calcula el promedio de todos los puntajes.
* **15-groups.sql**: Agrupa y cuenta cuántos registros hay por cada puntaje.
* **16-no_link.sql**: Lista nombres y puntajes, filtrando los valores nulos.

---
*Nota: Los scripts están diseñados para ejecutarse en un servidor MySQL.*
